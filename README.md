# Widerrufsbelehrung (B2C)

Dies sind ist die Widerrufsbelehrung der Quantum Electronic Devices GmbH für Endkunden.

Das Musterwiderrufsformular basiert auf den gesetzlichen Vorgaben nach (Artikel 246a EGBGB Anlage 2)[https://www.gesetze-im-internet.de/bgbeg/art_253anlage_2.html].

## Fragen
Für Fragen stehen wir Ihnen gerne zur Verfügung. Bitte wenden Sie sich per [E-mail](mailto:legal@quantum-electronic-devices.de) an uns.

## Versioning
Wir nutzen [SemVer](http://semver.org/) zur Versionierung. Alle Versionen dieses Textes finden sind unter [tags](../../tags) zu finden.

- MAJOR-Versionen spiegeln signifikante Änderungen wider, die beispielsweise auf neuen Gesetzen basieren.
- MINOR-Versionen spiegeln kleinere Änderungen wider, wie beispielsweise Klarstellungen.
- PATCH-Versionen beinhalten keinerlei inhaltliche Änderungen, sondern ausschließlich Korrekturen von Tippfehlern oder Stiländerungen.
